# Bin2Dec

An iOS app which takes the user input as binary format (**consisting of 0's and 1's**) and converts the binary number entered by the user to decimal equivalent.

**Language Used:** Swift 5

**Highlights:** 
  1. UIKit 
  2. Auto Layout - Constraints 
  3. Unit Testing 
  4. MVC
